Методика использования материалов ВЛС для картографирования рельефа

Общая схема процесса обработки данных ВЛС

Для того чтобы получить из материалов ВЛС различные морфометрические карты, следует обозначить последовательность операций: выделение из облака точек те, которые относятся к поверхности земли; создание цифровых моделей рельефа на основе выделенных точек; создание производных моделей рельефа, отражающих его различные морфометрические особенности; построение на основе моделей морфометрических карт.

Последовательность операций

Каждый из этапов может быть выполнен различными способами, и между этими способами есть разница с точки зрения получаемого результат. Методы в каждом этапе имеют преимущества или недостатки относительно других, для выявления наиболее эффективного и подходящего для установленной задачи, имеющихся исходных данных и ожидаемого результата. В последующих параграфах будут описаны возможные методы для каждого из определѐнных этапов.

Рассмотрение методов выделения точек земной поверхности по данным ВЛС

Для того чтобы построить по данным ВЛС карты рельефа, сначала нужно определить, какие точки из облака точек относятся к поверхности земли. Существует несколько способов восстановления рельефа: метод абсолютного минимума, метод конуса, молдинг и метод анализа пространственного спектра [4].

Метод абсолютного минимума — самый простой метод из существующих. Основан он на том, что строится сетка квадратов, которые захватывают определенную группу лазерных точек. Из этой группы выбирается одна точка, имеющая наименьшую геодезическую высоту.

Метод конуса основан на принципе, что из каждой точки облака стоится конус с вертикальной осью. Соответственно, вершины конусов, попадающие внутрь других конусов, исключаются. Оставшимся точкам присваивается атрибут земной поверхности.

Метод молдинга

слева: развитие триангуляционной сети
справа: метод в профиле (N - фиксированный размер ячейки)

Концепция молдинга (molding) основана на: итерационный принцип построения поверхности земли с постоянным контролем сходимости; использовании триангуляционной формы представления; использовании некоторого набора аналитических ограничений, отражающих гладкость поверхности «истинной земли».

На первом этапе строится грубая триангуляционная модель начального приближения. Это может быть выполнено на основе метода абсолютного минимума, то есть выделения точек с минимальной высотой. Далее выполняется развитие сети, то есть включение новых точек в качестве узлов сети. В данном случае при включении новой точки в сеть, исключаются старые связующие ребра, появляются новые.

Чтобы точка была включена в сеть, должны быть выполнены следующие условия:

1. После включения новой точки аппроксимация сохраняется;

2. Угол между гранями не должен превышать заранее установленное пороговое значение;

3. Добавление точки и изменение модели должно удовлетворять условию Делоне.

Одним из назначений молдинга является то, что при построении поверхности рельефа, выполняется выделение тех точек, которые принадлежат этой поверхности. Из недостатков стоит выделить значительные временные затраты, а так же необходимость знать и устанавливать значения параметров, например угол между гранями.

Метод анализа пространственного спектра является на сегодняшний день наиболее точно из всех методов представляющим истинную поверхность земли. Он включает в себя следующие этапы: инициализация набора точек, относящихся к рельефу; спектральный анализ выбранного набора точек и составление аппроксимирующей поверхности для отфильтрованных точек; определение условий набора группы новых точек из смежных областей по результатам выполнения имеющейся итерации. Предполагается, что искомая поверхность имеет пространственный спектр конечной ширины везде, кроме конечного числа так называемых структурных линий, на который претерпевает скачок либо сама функция, либо ее первая производная.

С использованием аналитической функции с ограниченным спектром для отобранного множества точек выполняется аппроксимация выбранного набора точек. По сути, выполняется фильтрация. Это имеет ряд благоприятных последствий. Можно выполнить выборку лазерных точек внутри множества точек-претендентов, приняв как критерий расстояние от найденной поверхности. В процессе отсеивания точек можно повторно провести аппроксимацию, тем самым улучшая оценку функции поверхности s(x, y). Полученная функция представляет собой математически «гладкое» отражение поверхности рельефа. Определить точность полученной модели можно с помощью средней квадратической ошибки точек, которые были использованы для аппроксимации поверхности. Методы построения цифровой модели рельефа

Цифровая модель рельефа (ЦМР) — форма представления данных и способ их структурного описания, позволяющий восстанавливать объект путем интерполяции, аппроксимации или экстраполяции. Для создания ЦМР существуют различные алгоритмов и методы интерполяции.

Триангуляция с линейной интерполяцией (Triangulated Irregular Network,TIN). При использовании данного метода точки могут размещаться как регулярно, так и нерегулярно. Поверхность строится путем соединения точек ребрами, образующими смежные треугольные грани, не накладываясь друг на друга, то есть триангуляцией.

Метод сплайнов (Spline), позволяет оценивать полученные значения отметок, используя математическую функцию, которая минимизирует полное поверхностное искривление. В итоге получается гладкая поверхность, которая проходит точно через исходные пункты. Она похожа на резиновый лист, то есть поверхность проходит через пункты при уменьшении полного искривления поверхности. Метод является лучшим для представления гладко изменяющихся поверхностей явлений. В современных программных средствах ГИС применяются две разновидности метода сплайнов: регуляризованный (плоский) сплайн и сплайн с натяжением. Плоский сплайн позволяет создавать гладкую, постепенно меняющуюся поверхность. В сплайне с натяжением используются только вторые производные, это включает большее количество исходных пунктов в вычислениях, которые обычно создают более гладкие поверхности, но вычисления занимают больше времени.

Диаграмма Вороного для заданного множества точек – это такое разбиение плоскости на совокупность областей, при котором любая точка в отдельной области разбиения ближе к одной из точек множества, чем к любой другой точке множества. Диаграммы Вороного также называют разбиением Тиссена или ячейками Дирихле. Одним из главных свойств диаграммы Вороного является еѐ взаимно-однозначное соответствие с триангуляцией Делоне. А именно, если соединить рѐбрами исходные точки, чьи многоугольники (ячейки) Вороного граничат друг с другом, то получим триангуляцию Делоне. Методом естественного соседа можно эффективно обрабатывать большой набор исходных данных. В нем выбираются ближайшие узлы, которые формируют поверхность вокруг пункта интерполяции. Метод имеет много положительных особенностей, а именно может использоваться как для интерполяции так и для экстраполяции, и вообще хорошо работает со сгруппированными и рассеянными пунктами.

Метод кригинга (Kriging) является статистический методом интерполяции. Кригинг предполагает, что расстояние или направление между типовыми пунктами отражают пространственную корреляцию, которая может использоваться, чтобы объяснить изменение в поверхности. Этот подразумевает приближение функции к указанному числу пунктов или всех пунктов в пределах указанного радиуса, чтобы определить значение для каждой точки. Преимущественно метод используется для построения поверхности, когда известны пространственно-коррелированное расстояние или тенденция в изменении значений данных. В основном применяется в геологии и науке о почвах.

Расчет морфометрических параметров

Морфометрические параметры отражают характеристики поверхности рельефа необходимые для анализа. Часть морфометрических алгоритмов работает на основе операции окрестности. По сути, строится регулярноячеистая матрица, которая по всей поверхности растра повторяет математическую формулу для каждой ячейки растра. Внутри этой матрицы определяется степень изменения показателя центральной ячейки относительно соседних.

Для построения моделей, отражающих морфометрические характеристики рельефа, требуется расчет соответствующих морфометрических показателей, таких как крутизна склона (угол наклона), экспозиция склона и кривизна поверхности рельефа в плане и в профиле. Кривизна в профиле может назваться профильной кривизной или вертикальной кривизной, а кривизна в плане – горизонтальной кривизной [9, 2].

ЦМР дискретна, а не непрерывная поверхность, следовательно нужно рассчитывать локальные морфометрические параметры «по определению». Для их расчета необходимы специальные математические методы. У каждой ячейки есть ее значение высоты, плюс значение из соседних ячеек скользящей матрицы. Простой подход к вычислению углов наклона, это использовать фильтры x- и y-градиентов. Фильтр Х-градиента определяет коэффициент увеличения наклона поверхности с запада на восток. Например, если высота к западу от центральной ячейки матрицы составляет 150 метров, а к востоку от центральной ячейки – 162 метра, то вдоль этого направления высота увеличивается на 12 метров на две ширины ячейки, то есть x-градиент равен 6 метр на ширину ячейки. Фильтр y-градиента работает, но в направлении юг-север. Оба фильтра выражают увеличение высоты на ширину ячейки. Следующим шагом в анализе локальных морфометрических переменных является метод с использованием вторых производных. Они связаны с вогнутостью и выпуклостью поверхности. Параметр, описывающий это, называется кривизной. Важным понятием для понимания геометрического смысла кривизны является так называемый нормальный участок гладкой поверхности.
