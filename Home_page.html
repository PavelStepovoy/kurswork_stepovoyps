ГЕОИНФОРМАЦИОННОЕ КАРТОГРАФИРОВАНИЕ РЕЛЬЕФА ПО МАТЕРИАЛАМ ВОЗДУШНОГО ЛАЗЕРНОГО СКАНИРОВАНИЯ

Лазерное сканирование является одной из перспективных технологий дистанционного зондирования Земли. Данный метод имеет широкие технологические возможности, которые можно реализовать в различных направлениях деятельности человека. Лазерное сканирование позволяет определить пространственную структуру объекта, в точности передовая его форму. Такое качество важно для отображения неровностей поверхности Земли – рельефа.

По методике сбора данных лазерное сканирование подразделяется на несколько видов: наземное, мобильное и воздушное. Для географического картографирования рельефа наибольший интерес представляет воздушное лазерное сканирование.
